1.package control

(1)通过快捷键 ctrl+` 或者菜单（View > Show Console）
打开控制台，然后粘贴下列的 Python 代码

import urllib.request,os,hashlib; h = ‘7183a2d3e96f11eeadd761d777e62404’ + ‘e330c659d4bb41d3bdf022e94cab3cd0’; pf = ‘Package Control.sublime-package’; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( ‘http://sublime.wbond.net/‘ + pf.replace(‘ ‘, ‘%20’)).read(); dh = hashlib.sha256(by).hexdigest(); print(‘Error validating download (got %s instead of %s), please try manual install’ % (dh, h)) if dh != h else open(os.path.join( ipp, pf), ‘wb’ ).write(by)

另外提供Sublime text 2的Package Control的安装代码:

import urllib2,os,hashlib; h = ‘7183a2d3e96f11eeadd761d777e62404’ + ‘e330c659d4bb41d3bdf022e94cab3cd0’; pf = ‘Package Control.sublime-package’; ipp = sublime.installed_packages_path(); os.makedirs( ipp ) if not os.path.exists(ipp) else None; urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler()) ); by = urllib2.urlopen( ‘http://sublime.wbond.net/‘ + pf.replace(‘ ‘, ‘%20’)).read(); dh = hashlib.sha256(by).hexdigest(); open( os.path.join( ipp, pf), ‘wb’ ).write(by) if dh == h else None; print(‘Error validating download (got %s instead of %s), please try manual install’ % (dh, h) if dh != h else ‘Please restart Sublime Text to finish installation’)

(2)手动安装

a.点击Preferences > Browse Packages

b.上面会打开一个文件夹,在其中找到Installed Packages文件夹(如果没找到,在父击级目录中找)

c.在插件集合上下载Control.sublime-package,然后拷贝到Installed Packages文件夹

d.重启sublime

(3)如果在preferences-package settings中看到package control这一项,则表明安装成功

注:
(1)有时候ctrl+` 打不开控制台,可能会与其他的快捷键冲突,这是可以在Preferences -> Key Bindings – Default中修改:

{ “keys”: [“ctrl+backquote”], “command”: “show_panel”, “args”: {“panel”: “console”, “toggle”: true} }

把ctrl+backquote修改为其他的快捷键就行了。

(2)Installed Packages和Packages文件夹

只有单一文件（插件名.sublime-package）的插件，被放在了Installed Packages目录中，包含有多个文件的安装包插件（比如ConvertToUTF8，IMESupport），被放在了Packages 目录中。

计算机中的内存是以位为最小存储单位的。通过对内存进行组织，可以引用特定的位集合。把计算机的内存起始位编号为1，每隔8位编号增1，也就是以字节为单位，每隔一个字节编号向上加一，可以对计算机所有内存进行编号。

字符’1’只是代表一个形状为’1’的符号，在需要时按原样输出，在内存中以ASCII码形式存储，占1个字节。

整数1是以整数存储方式(二进制补码方式)存储的，占2个或4个字节。

如何存储和如何表示的是不一样的。存储都以补码形式存储。

例子:

int a = 1;

Printf(“%d,%c”,a,a); //%d是输出a数值本身。%c是输出ASCII码值为a的字符

char ch = ‘\0’;

Printf(“%d,%c”,ch,ch); //%d是求它的编号(ASCII码编号), %c是求ASCII码字符本身

字符型变量在内存中占一个字节，由于存储的是字符的二进制ASCII码(补码?)，与整型数据存储方式类似，字符型数据和整型数据可以相互运算。

注意： 字符串常量不允许赋值给字符型变量，C语言也没有专门的字符串变量。要在内存中存取字符串，只能使用数组或指针。
